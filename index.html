Удобный интерфейс, управление одним пальцем

Приложение выдает в произвольном порядке слова из Google Таблицы. Пользователь нажатием на верхнюю часть экрана (зеленое или красное поле) постепенно открывает слово. Нажатием на серое поле переходит к следующей паре слов. Цвета приглушены и подобраны так, чтобы не утомлять глаза при долгом обучении.

Подходит как для латинских символов, так и для иероглифов

Лучший подход при обучении - давать ответ не сразу, начать с подсказки. В приложении раскрытие слов можно настроить: под латинские слова при нажатии раскрывается 3 символа, при выборе иэроглифов символы раскрываются по одному. Так достигается одинаковый опыт использования, без лишнего утомления.

Тренажер, который усиливает запоминание

Главная особенность приложения - механизм выдачи слов из словаря. Пользователь выбирает новую группу слов (прибавляет по 10 слов каждый урок) - дальше к ним добавляются слова ранее выученные, для закрепления материала. Предыдущие слова появляются с меньшей вероятностью, группами из занятий 7, 14, 21 и т.д. уроков назад. Визуализация точных промежутков выводится в меню настроек. Также настройки позволяют скрывать при выдаче карточек только перевод слов, или только оригинал, или скрывать их произвольно.
